
JS –язык с динамической типизацией

В JS 8 основных типов данных.

"Примитивные" типы:
1) number
	числовой тип данных (и целые, и с плавающей точкой). Ограничен диапазоном `±(253-1)`
2) bigint
	для целых чисел произвольной длины.
3) string
	Строка может содержать ноль или больше символов, нет отдельного символьного типа.
4) boolean
	для `true`/`false`
5) null
	для неизвестных значений – отдельный тип, имеющий одно значение `null`.
6) undefined
	для неприсвоенных значений – отдельный тип, имеющий одно значение `undefined`.
7) symbol
	для уникальных идентификаторов.
Не "примитивный"
8) object
	для более сложных структур данных.
