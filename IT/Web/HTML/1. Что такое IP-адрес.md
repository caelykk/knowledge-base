# Что такое IP-адрес?

Объединение компьютерных сетей в единую всемирную структуру стало возможным благодаря **<font color="#ebd96c">протоколу IP</font>**, реализующему адресную систему, в которой каждому компьютеру присваивается индивидуальный адрес (**<font color="#ebd96c">IP-адрес</font>**, состоящий из 4 групп цифр, разделенных точками).
