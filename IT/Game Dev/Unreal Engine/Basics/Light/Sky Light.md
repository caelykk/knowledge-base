1. Захватывает удаленные части уровня
2. Применяется к сцене в виде света (отражений)
3. Глобальное освещение
4. "Захватывает" только определенные условия:
	- Для статичных источников света (Static Lights), обновляется при создании освещения
	- Для стационарного (Stationary) и передвижного (Movable) освещения, обновления выполняются один раз при загрузке и при "захвате"
	- Постоянно, когда включена запись в режиме реального времени
