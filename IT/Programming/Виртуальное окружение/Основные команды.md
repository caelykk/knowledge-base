Создание виртуального окружения
<font color="#d83931">$</font> <font color="#00b050">python -m venv name_venv</font>
  
Активация виртуального окружения  на Windows  
<font color="#d83931">$</font> <font color="#00b050">venv\Scripts\activate.bat  </font>
  
Посмотреть установленные пакеты  
<font color="#d83931">$</font> <font color="#00b050">pip freeze  </font>
  
Установить необходимую библиотеку/модуль  
<font color="#d83931">$</font> <font color="#00b050">pip install name_module</font>  
  
Обновить библиотеку/модуль  
<font color="#d83931">$</font> <font color="#00b050">pip install -U pip name_module</font>  
  
Удалить установленную библиотеку/модуль  
<font color="#d83931">$</font> <font color="#00b050">pip uninstall name_module -y  </font>
  
Создаем файл с зависимостями  
<font color="#d83931">$</font> <font color="#00b050">pip freeze > requirements.txt  </font>
  
Установить зависимости из файла  
<font color="#d83931">$</font> <font color="#00b050">pip install -r requirements.txt</font>  
  
Отключить виртуальное окружение  
<font color="#d83931">$</font> <font color="#00b050">deactivate</font>
