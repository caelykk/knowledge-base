Если говорить кратко, то `итератор` — это механизм поэлементного обхода данных, а `генератор` позволяет отложено создавать результат при итерации. 

В частном случае `генератор` вообще является подтипом `итератора`. Но не наоборот.
```python
import typing _gen = (i for i in range(5)) # генератор 
_iter = iter(range(5)) # итератор 

print(_gen) 
print(_iter) 

print(issubclass(_gen.__class__, typing.Iterator)) print(issubclass(_iter.__class__, typing.Generator))
```
