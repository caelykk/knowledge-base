## Точка зрения программиста

Программа на Python представляет собой всего лишь тек­стовый файл, содержащий операторы Python. Например, файл по имени `script0.ру` находится простейший сценарий Python, но он является полнофункциональной программой Python:

```python
print(’hello world’)
print(2 ** 100)
```

Файл `scriptO.py` содержит два *Python-оператора* <font color="#ebd65a">print</font>, которые выводят в вы­ходной поток строку (текст в кавычках) и результат числового выражения (2 в степе­ни 100).Файл scriptO.py содержит два Python-оператора print, которые выводят в вы­ходной поток строку (текст в кавычках) и результат числового выражения (2 в степе­ни 100).

Мы можем создать такой файл с операторами в любом текстовом редакторе, кото­рый вам нравится. По соглашению файлам с программами Python назначаются имена, оканчивающиеся на `.ру`; формально такая схема именования обязательна только для файлов, которые “импортируются”, но ради согласованности большинство файлов Python имеют имена `.ру`.

После набора всех операторов в текстовом файле вы должны сообщить интерпре­татору Python о необходимости `выполнить` файл — что просто означает запуск всех операторов в файле сначала до конца, друг за другом.

Если выполнение файла проходит гладко, тогда вы увидите результаты двух операторов <font color="#ebd65a">print</font> в каком-то месте — по умолчанию в том же окне, где запускалась программа:

```python
hello world
1267650600228229401496703205376
```
