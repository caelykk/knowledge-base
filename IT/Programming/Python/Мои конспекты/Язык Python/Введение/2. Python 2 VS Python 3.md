## Python 2 VS Python 3

Основные версии языка Python – **<font color="#c3d69b">Python 2</font>** и **<font color="#b2a2c7">Python 3</font>**. Версия **<font color="#c3d69b">Python 2</font>** считается устаревающей, версия 3 — более новой и современной. 

## Почему не откажутся от второй версии?

Если коротко, **<font color="#b2a2c7">Python 3</font>** не имеет полной обратной совместимости с предыдущей версией, а на **<font color="#c3d69b">Python 2</font>** написано очень много программ. У разработчиков нет возможности переписать всё на новую версию.
