## Язык Python

Язык **<font color="#d99694">Python</font>** разработал голландский программист **Гвидо Ван Россум** (Guido van Rossum) в 1991 году. Гвидо был фанатом британского комедийного сериала «Monty Python’s Flying Circus», откуда и пришло название языка.

**<font color="#d99694">Python</font>** - мощный компьютерный язык программирования, поддерживающий множество парадигм, который оптимизирован для обеспечения высокой продуктивности программистов, читабельности кода и качества программного обеспечения.

<font color="#d99694">Python</font> - это популярный язык программирования с открытым исходным кодом, применяемый для написания автономных программ и сценарных приложений.
