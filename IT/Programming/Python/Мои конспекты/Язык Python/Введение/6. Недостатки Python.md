## Недостатки Python

1. **<font color="#c3d69b">Низкая скорость выполнения</font>** по сравнению с такими языками, как C и C++;
2. <font color="#c3d69b">Динамическая типизация языка</font> — минус при написании сложных программ.


Единственный значительный недостаток текущей реали­зации Python связан с тем, что скорость выполнения может не всегда быть такой же, как у полностью компилируемых и низкоуровневых языков, подобных С и C++.

Современные стандартные реализации Python `компилируют` (т.е. транслируют) операторы исходного кода в промежуточный формат, известный как <font color="#d99694">байт-код</font>, и затем `интерпретируют` этот <font color="#d99694">байт-код</font>. <font color="#d99694">Байт-код</font> обеспечивает переноси­мость, т.к. он представляет собой независимый от платформы формат. Тем не менее, поскольку код Python обычно не компилируется до машинного кода (например, до инструкций для процессора Intel), некоторые программы будут выполняться в Python медленнее, чем в полностью компилируемом языке наподобие С.
