## Разработка и поддержка

Разработчики Python координируют работу в онлайновом режиме с помощью системы управления версиями. Изменения разрабатываются согласно формальному протоколу, который включает написание <font color="#c3d69b">предложения по улучшению Python</font> (<font color="#d99694">Python Enhancement Proposal</font> — <font color="#d99694">PEP</font>) или другого документа и расширение системы регрессионного тести­рования Python.

<font color="#c3d69b">Фонд программного обеспечения Python</font> (<font color="#d99694">Python Software Foundation</font> — <font color="#d99694">PSF</font>), официаль­ная некоммерческая группа, организует конференции и занимается вопросами интел­
лектуальной собственности. Многочисленные конференции, посвященные Python,
проводятся по всему миру; крупнейшими считаются OSCON, которую организует
O’Reilly, и РуСоп, проводимая PSF. Первая касается множества проектов с открытым
кодом, а вторая представляет собой событие, связанное только с Python, популяр­ность которого в последние годы значительно возросла.
