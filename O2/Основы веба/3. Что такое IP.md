
### IP

**IP (Internet Protocol)** — это протокол, который используется для адресации устройств в сети Интернет и обмена данными между ними. **IP** позволяет компьютерам и другим устройствам находить друг друга и взаимодействовать.

Когда мы вводим **URL** в браузер, система **DNS** преобразует доменное имя в **IP-адрес**, который указывает на конкретное устройство в сети.

### IP-адрес

**IP-адрес** — это уникальный числовой идентификатор, который присваивается каждому устройству в сети (например, компьютеру, серверу, маршрутизатору и т.д.). С помощью IP-адреса устройства могут обмениваться данными в сети.

Каждое устройство, подключённое к сети, должно иметь уникальный IP-адрес, чтобы другие устройства могли найти его и отправить туда данные.

### Типы IP-адресов

1. **IPv4** (Internet Protocol version 4):
	- Это наиболее распространённая версия IP, используемая в Интернете.
	- **Формат:** состоит из четырёх 8-битных чисел (от 0 до 255), разделённых точками, например: `192.168.1.1`
	- В IPv4 доступно **4,3 миллиарда** (2^32) уникальных адресов (но из-за роста сети их оказалось недостаточно).


2. **IPv6** (Internet Protocol version 6):
	- Введён для решения проблемы нехватки адресов в IPv4.
	- **Формат:** состоит из восьми групп шестнадцатеричных чисел (от 0 до F), разделённых двоеточиями, например: `2001:0db8:85a3:0000:0000:8a2e:0370:7334`
	- IPv6 позволяет создать **340 ундилиона** (3,4 × 10^38) уникальных адресов, что делает его почти безлимитным в плане доступных адресов.

### Как работает IP?

1. Когда ты вводишь в браузере URL, например, `https://www.example.com`, система DNS ищет соответствующий **IP-адрес** для `www.example.com`.
2. DNS возвращает, например, IP-адрес `93.184.216.34`.
3. Браузер использует этот IP для соединения с сервером, который находится по этому адресу.
4. Через установление соединения (например, через TCP) происходит обмен данными: сервер отправляет HTML-контент, и браузер отображает страницу.
